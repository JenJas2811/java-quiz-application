# Java Quiz Application

This is a simple desktop quiz application built using Java Swing and MySQL.

## Features
- Multiple choice quiz system
- Graphical User Interface using Java Swing
- Score calculation
- MySQL database storage
- Result display at the end

## Technologies Used
- Java
- Swing
- MySQL
- JDBC

## How It Works
1. User starts the quiz.
2. Questions are displayed with four options.
3. User selects the answer and clicks Next.
4. At the end, the score is calculated.
5. Score is stored in the MySQL database.
